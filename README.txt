> Prueba MaxiVideo:

El siguiente proyecto está basado en una prueba para demostrar habilidades técnicas a la hora de desarrollar.
El proyecto consiste en diseñar e implementar una app web donde el usuario final tendrá la posibilidad de 
alquilar un ejemplar de películas o modificar dicho alquiler por medio de un formulario que estará conectado
a una base de datos local.

## Información general

Este proyecto se llevó a cabo con la finalidad de presentar mis habilidades en desarrollo a los evaluadores y 
reclutadores del grupo asistencia.
La prueba se llevó a cabo en su totalidad y completa funcionalidad.
 



## Tecnologías utilizadas
- Lenguajes usados:
  *Hacia la parte del Frontend se utilizo html5, css, js, con el apoyo 
   de bootstrap 5 como framework de estilos.

  *Hacia la parte del Backend se utilizo como lenguaje base Php y Mysql
   para la creación y modificación de la base de datos.
 
- Editor de Código: Visual Studio Code - versión 1.68.0 for Windws

- Servidor Local: Xampp v 3.3.0


## Características

El proyecto esta implementado por 4 funciones base que nos permiten hacer el CRUD en
la base de datos y adicional una interfaz gráfica donde el usuario podrá hacer uso 
de estas funciones. Se inicia el proyecto con datos base para la prueba del correcto
funcionamiento y testigo.





## Uso
- El proyecto contendrá 2 carpetas principales:

1) En la primera carpeta llamada Base_De_Datos podremos encontrar todos los archivos relacionados con la
base de datos:
- Un archivo .Sql que contine toda la base de datos
- Un archivo .pdf que contiene el Modelo Entidad Relación de la base de datos
- Un Archivo .xlsx que contiene los datos de las tablas iniciales de manera ordenada

2) En la segunda carpeta llamada Proyecto_MaxVideo vamos a encontrar todo el código fuente para
ser ejecutado en un editor preferiblemnte en Visual Studio Code

Para el correco funcionamiento de proyecto debemos como primera instancia crear la base de datos por medio del
archivo .Sql, el paso a seguir seria utilizar un servidor local como Xampp y verificar que los puertos estén
habilitados para dar inicio al servidor apache y seguidamente a Mysql. Después de estos dos pasos ya podemos
abrir el proyecto con el código fuente y verificar que la  conexión entre php y mysql este correcta, como 
ultima instancia ya podemos correr el código y hacer uso de nuestra app.
